# OpenCV SVM Machine Learning Tutorial

Repository ini berisi contoh implementasi **Machine Learning menggunakan Support Vector Machine (SVM)** dengan Python dan OpenCV.

Project ini dibuat sebagai materi pembelajaran untuk memahami dasar penggunaan SVM pada Computer Vision.

---

## 📚 Materi yang Dipelajari

Dalam notebook ini dijelaskan beberapa konsep penting:

- Import library Python
- Membuat dataset training sederhana
- Visualisasi dataset
- Konfigurasi model SVM
- Training model machine learning
- Testing model
- Penjelasan kode baris per baris

---

## 🛠 Teknologi yang Digunakan

- Python
- OpenCV
- NumPy
- Matplotlib
- Jupyter Notebook
- VsCode

---

## 🚀 Cara Menjalankan Project

1. Clone repository ini


git clone (https://github.com/Tasri22/Tugas_Kelompok_6_ML.git)


2. Masuk ke folder project


cd opencv-svm-tutorial


3. Install dependency


pip install opencv-python numpy matplotlib


4. Jalankan Jupyter Notebook


jupyter notebook


5. Buka file:


OpenCV_6.ipynb
